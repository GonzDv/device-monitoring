# DeviceMonitoring

Plataforma on-premise de monitoreo de equipos vía SNMP (impresoras y red).

## Estructura
- `backend/`  — API y poller (Python / FastAPI)
- `frontend/` — Dashboard (React)
- `infra/`    — Docker Compose y configuración
- `docs/`     — PRD, arquitectura y decisiones
